## chapter 1
1.5节的类简介其实挺神奇的，比如std::cin可以读入到基本变量类型，但这里却可以读入自定义类型：
```
Sales_item book;
std::cin >> book;
```
当然后面我们知道了，是因为重载了>>运算符。
